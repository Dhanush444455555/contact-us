# Contact Us React App

A beautiful contact us page built with React and Vite, featuring grand animations.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:5173](http://localhost:5173) in your browser.

## Features

- Responsive contact form
- Contact information display
- Social media links
- Grand CSS animations
- Gradient backgrounds
- Hover effects

## Technologies Used

- React
- Vite
- CSS3 Animations
