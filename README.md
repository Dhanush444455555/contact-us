# Contact Us - Siddaganga Institute of Technology

Modern React-based contact form with GPS map integration and grand animations.

## Features
- 📝 Contact form with validation
- 🗺️ Google Maps integration showing institute location
- ✨ Grand CSS animations
- 🎨 Dark theme with neon colors
- 📱 Fully responsive design
- 💫 Smooth transitions and effects

## Quick Start

### Run Locally
```bash
npm install
npm run dev
```
Visit `http://localhost:5173`

### Build for Production
```bash
npm run build
npm run preview
```

## Tech Stack
- React 18
- Vite
- CSS3 Animations
- Google Maps API

## Project Structure
```
src/
├── App.jsx          # Main component
├── main.jsx         # React entry point
├── ContactUs.css    # Styles & animations
└── index.css        # Base styles
```
